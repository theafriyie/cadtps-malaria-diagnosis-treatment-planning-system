# Treatment Planning

This folder contains materials related to treatment recommendation and follow-up support in the CADTPS project.

## Treatment Planning Factors

- age
- drug allergies
- previous history
- diagnosis result
- severity level

## Features

- treatment plan generation
- medication schedule guidance
- dosage support
- safety warning support
- follow-up reminders
- doctor alerts
- patient medication reminders

## Goal

The treatment planning module is intended to support safer and more structured malaria care after diagnosis.
