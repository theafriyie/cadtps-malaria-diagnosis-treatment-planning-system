# Diagnosis Logic

This folder contains notes and materials related to the diagnosis support workflow of the CADTPS project.

## Diagnosis Inputs

- patient age
- symptoms
- medical history
- laboratory test results

## Diagnosis Outputs

- malaria diagnosis support
- malaria type prediction support
- severity classification:
  - mild
  - moderate
  - severe

## Goal

The goal is to help healthcare providers make faster and more informed diagnostic decisions using structured digital support.
